# F1 Simulator (backend)

## Installation

```bash
$ yarn
```

## Building the app

```bash
$ yarn build
```

## Running the app

development:
```bash
$ yarn start
```
watch mode:
```bash
$ yarn start:dev
```
production mode:
```bash
$ yarn start:prod
```

## Accessing the app

[http://localhost:3000](http://localhost:3000) or the custom port you define in dotenv like [:1337](http://localhost:1337) or so.
